實用、有趣網站與工具大集合

雲端

1. pCloud

免費最高可達10GB(需做任務)

2. PikaPods

  • 簡易自架伺服器，Kavita也可以用這個一鍵架設與部署
  • 收費
  • Code server、Jupyter、shiori etc.

有趣類

TypeRacer打字比賽

中打有奇怪的全形引號，打不出來😥

AI Emoji Generator

  • 透過prompt讓他產生自己想要的組合Emoji!

設計類

VEED.IO

一家英國新創公司，致力於打造直覺、簡易的影片剪輯軟體

Canva

  • Online剪輯拼貼圖片等等

CakeResume

  • 履歷範本

Pika

  • 文字轉影片
  • 兩位史丹佛大學博士合作開發的project
  • 蓬勃發展中

閱讀

1. Kavita

Kavita is a fast, feature rich, cross platform reading server.

  • 可閱讀漫畫、書籍等
  • self-hosted

2. BookFusion

  • 免費十本書上傳
  • 付費無限制，一個月1.99美金
  • 多平台

GitHub

FiraCode

  • 適合工程師的字體
  • 可以連字!
  • 等寬
  • 易分辨I或l等等易混淆字母

Webpilot

  • 強大的工具讓chatGPT可連網
  • 目前chrome extension無法訪問及下載

pipx

  • 在獨立環境安裝並執行python程式
  • 可以把每個python程式和其依賴包都獨立分開，就不會有衝突的問題
  • 注意這個工具安裝的是end-user application，所以跟寫python時import package是不一樣的
  • 安裝如果出現找不到tomli的話直接pip install tomli就好

cowsay-python

  • 趣味小工具，print的時候有多種character，例如牛、豬、企鵝等等的動物可以選擇，print的資訊會顯示在對話氣泡

dooit

  • 基於textual並使用python開發的CLI代辦清單工具

Frogmouth

  • 同樣是基於textual開發的Markdown閱讀器

工具類

SimilarSites

  • 找某個網站的相似網站

Sci-hub

  • 找免費論文

AlternativeTo

  • 某個軟體的相似軟體

Ai Tool Hunt

  • 用AI找AI工具

Vercel

  • 免費網頁部署

Replicate

  • 雲端執行各種AI model，例如把stable diffusion傳上去，就可以使用他們的硬體來產生圖片
  • 須付費

Squoosh

  • 線上圖片壓縮

Coolors

  • 挑選主題色
  • 可用來幫網站挑選配色
